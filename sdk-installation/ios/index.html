Getting Started

iOS SDK Installation

  1. When using Cocoapods, add the Smartlok Pod to your Podfile.
pod Smartlook

otherwise, download the latest Smartlook iOS SDK, unzip the file and add Smartlook.framework to your Xcode project.

  1. Add Smartlook to your AppDelegate among other imports, then setup and start recording in the application's didFinishLaunching method:
swiftobjective-c
import Smartlook
...
func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    Smartlook.setup(key: "YOUR_API_KEY")
    Smartlook.startRecording()

Supported versions

Current minimal supported version is iOS 10.

Important things to know

  • To setup Smartlook, you need your unique API key. This can be found in your mobile project settings in your Smartlook Dashboard.
  • SDK will upload sessions only on WiFi if you want to upload sessions on mobile connections too, you need to enable this in your project setting in the dashboard. For more details about when your recordings appear in the Dashboard, refer this detailed document.
  • Full API reference: Smartlook API reference
  • If you are experiencing any technical difficulties with our SDK feel free to report a issue to our issue tracker.
